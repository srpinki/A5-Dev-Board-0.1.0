Question-1: What are the different ways to select an element in the DOM?

There are several ways to select an element in the DOM. Here is the methods document. getElementById(), document. getElementsByClassName(), document. getElementsByTagName(), document. querySelector() & document. querySelectorAll().

Question-2: What is the difference between innerHTML, innerText, and textContent ?

"innerHTML" retrieves the complete HTML content within an element, including tags, while "innerText" only retrieves the visible text content of an element (ignoring HTML tags), and "textContent" retrieves all text content within an element, including hidden text, but without interpreting any HTML tags;

Question-3: What is event delegation in the DOM?

Event Delegation is a technique in JavaScript where instead of attaching event listeners to multiple child elements, We attach a single event listener to a common parent element.
